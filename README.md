Supply Chain Delivery Performance Analysis

This project provides an end-to-end analysis of supply chain data to identify bottlenecks, analyze delivery performance, and predict potential delays using Machine Learning.

📋 Project Overview
This project explores a real-world dataset to understand the impact of shipping times on customer satisfaction and business profitability. It includes data cleaning, exploratory data analysis (EDA), and the development of a predictive model to identify high-risk orders.

🛠 Tech Stack
Language: Python
Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
Techniques: Feature Engineering, Time Series Analysis, Machine Learning (Random Forest), Data Visualization
📊 Key Findings
Delivery Delays: Over 54% of orders deviate from scheduled delivery windows, causing significant profit erosion.
Bottlenecks: Identified specific regions (e.g., Central Africa) and shipping modes (First/Second Class) as primary drivers of delays.
Predictive Insights: Developed a Random Forest model achieving 74% accuracy in flagging at-risk deliveries before they occur.

🚀 Recommendations
Automated Alerts: Deploy the predictive model to flag high-risk orders at the time of purchase.
Process Optimization: Resolve payment processing frictions and audit high-delay departments.
Seasonal Planning: Enhance capacity planning during peak months (August, October, December).
